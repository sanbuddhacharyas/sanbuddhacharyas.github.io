Braille-Dristi(vision)

Built a device that converts printed textbooks into braille format. It consists of a camera and processor which clicks the picture of the book and uses Optical character recognition (OCR) to converts images into text. The text is sent via UART communication in ASCII format to the braille printer.

Technologies used: Raspberry pi, Camera, Optical character recognition, Image processing, TTS, Python
